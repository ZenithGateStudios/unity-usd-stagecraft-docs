USD Stagecraft Changelog

All notable changes to USD Stagecraft are documented here.

0.1.5 2026-06-18

Changed

  • English package metadata (package.json descriptions) for Asset Store compliance (Section 2.4.1).
  • English XML documentation, comments, tooltips, and user-facing log messages across public C# APIs.

Notes

  • Publisher documentation: zenithgatestudios.github.io/unity-usd-stagecraft-docs

0.1.4 2026-06-13

  • Minor bug fixes.

0.1.2 2026-05-08

Added

  • ThirdParty/README.txt: attribution, NOTICE, and full Apache 2.0 text for redistributed OpenUSD 0.26.5, Intel oneTBB, plus MaterialX / usdMtlx notice.

Notes

  • When upgrading the OpenUSD SDK under Core/external/USD, refresh third-party notices and version strings as applicable.

0.1.1 2026-04-29

  • Minor bug fixes.
  • Minor update.

0.1.0 Early Access 2026-04-29

Added

  • Runtime loading of OpenUSD .usd, .usda, and .usdc files.
  • UsdLoader async and sync USD file loading API.
  • UsdMeshBuilder mesh conversion, UsdMaterialLoader material support, and UsdTextureCache texture caching.
  • UsdStagePreview Editor preview mode and UsdFileWatcher hot reload.
  • macOS Apple Silicon arm64 native plugin.

Known limitations

  • macOS Apple Silicon only in Early Access.
  • USD variants and payloads are partially supported.

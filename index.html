About

Independent Unity tooling studio

ZenithGate Studios is an independent studio focused on Unity tooling for games and visual production. We build editor extensions, runtime plugins, and pipeline utilities that help teams move 3D content into Unity with less friction.

Our experience spans Unity game development and video-oriented plugin work, including tools for scene import, preview, iteration, and production workflows. We prioritize clear documentation, predictable Unity integration, and practical features for artists and technical developers.

USD Stagecraft is our OpenUSD-focused package for Unity. It extends this work by helping creators inspect, load, and iterate on USD scene data inside Unity.

Asset

USD Stagecraft

Runtime loading

Load local .usd, .usda, .usdc, and .usdz files through the UsdLoader API.

Editor preview

Preview a USD stage in the Scene view without entering Play mode using UsdStagePreview.

Iteration workflow

Use file watching, diff reload, variants, animation controls, and edit layers for faster DCC-to-Unity iteration.

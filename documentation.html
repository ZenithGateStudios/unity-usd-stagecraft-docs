Documentation

USD Stagecraft 0.2.0

Load OpenUSD (.usd / .usda / .usdc / .usdz) files at runtime in Unity, preview them in the Editor, and iterate with file watching and diff reload.

Supported environments

Unity2022.3 LTS or later
macOSApple Silicon (M1 / M2 / M3), arm64 only
Windowsx64
Render pipelinesBuilt-in Render Pipeline and URP are the primary targets (HDRP planned for v0.4.0)

Installation

Get USD Stagecraft on the Unity Asset Store (v0.2.0 — Windows x64 and macOS Apple Silicon, Early Access $59).

Watch the v0.2 workflow demo on X: Blender to Unity hot reload.

  1. Open Window > Package Manager in Unity.
  2. Set the package source to My Assets.
  3. Select USD Stagecraft, then download and import it.
  4. Confirm the package appears under Packages/com.zenithgatestudios.usd-stagecraft/.

Optional samples can be imported from the Package Manager samples section.

Quick start: Editor preview

  1. Add an empty GameObject or use the Stage Preview sample scene.
  2. Add the UsdStagePreview component.
  3. Set File Path to a valid USD file.
  4. The USD hierarchy appears under the GameObject in the Scene view without entering Play mode.

Changing File Path, Shader Name, or Scale reloads the preview in the Editor.

Quick start: Runtime loading

Place runtime USD files under Assets/StreamingAssets/, then load them with UsdLoader.LoadAsync.

using UsdStagecraft;
using UnityEngine;

public class LoadUsdOnce : MonoBehaviour
{
    private LoadResult result;

    private async void Start()
    {
        result = await UsdLoader.LoadAsync("/absolute/path/to/scene.usda", "Standard", 0f);
        if (!result.IsSuccess)
        {
            Debug.LogError(result.ErrorMessage);
            return;
        }

        result.Root.transform.SetParent(transform, false);
    }

    private void OnDestroy()
    {
        if (result != null && result.IsSuccess)
        {
            UsdLoader.Unload(result);
        }
    }
}

Core workflows

Stage preview

Edit mode uses synchronous loading for preview. Play mode uses asynchronous loading through Load() or Auto Load On Start.

Hot reload

When a watched USD layer or referenced texture changes on disk, USD Stagecraft refreshes the stage or materials where applicable.

Variants and edit layers

Inspector controls expose USD variants and edit-layer operations without modifying the original root USD as the primary workflow.

Known limitations

  • macOS Apple Silicon (arm64) and Windows x64.
  • The native plugin is not notarized in the current Early Access release.
  • Very large USD files may load slowly or stress memory.
  • USD variants and payloads are partially supported.
  • HDRP fuller support is planned for v0.4.0.
